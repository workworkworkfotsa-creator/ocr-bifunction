OCR BiFunction — Politiques d'exécution

Qui tourne quand : sync = dans la requête (secondes) ; async_immediate = file continue du watchdog ; async_nightly = passe de nuit (--nightly). La ligne * = défaut pour toute catégorie sans ligne propre. « Override » = le client de l'API peut imposer son processing_mode. Dépôt : page upload · Revue : page revue.

Politique de non-conformité (métier)

Ce qu'un document PROUVÉ non conforme déclenche : block = cet upload est refusé (défaut) ; block_holder = refuse aussi les uploads SUIVANTS du même titulaire déclaré tant que la non-conformité est ouverte ; flag_and_continue = rien n'est bloqué, le document part en revue humaine avec le flag. La preuve (template, checks calculés, document retenu) part à la revue / compliance.
